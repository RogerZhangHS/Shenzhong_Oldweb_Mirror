郭慧清老师：与你相遇

郭慧清　2015-09-01

与你相遇

郭慧清老师在深圳中学2015－2016学年第一学期开学典礼上的讲话

2015年8月31日

 

尊敬的王校长、尊敬的各位同事、亲爱的同学们：

早上好！

首先，我有几句话送给同学们，这是以你们的口气表达的：

n  我用女孩最美的年华陪伴你

n  我用男孩最初的梦想拥抱你

n  今生让我遇见你

n  只因我在找寻你

我先问：你认为上述的“你”表示谁呢？

也许同学们会说：老师，就是你啊！

谢谢同学们！这是老师最大的幸福．

但是，我今天想说的，是那个“你”表示数学．

由此看来，“你”是一个变量，这是从数学的角度看世界的一个重要观点．

今天我演讲的标题是“与你相遇”，副标题是“怎样学好数学？”．我想，送给同学们的四句话，已经表达了我对怎样学好数学的基本观点．

为了学好数学，我们要先认识数学．

先看两幅图．

 

在图1中，“个”中的“I”表示垂直合拢放在胸前的两只手，如果坚持这种状态不愿改变，表示这个人不愿与别人联系，长期如此，“I”会将这个人撕裂，“个”就变成了“小”，这“个人”也就成了“小人”．

在图2中，由“个”到“大”，两只手水平打开了，这种状态表示这个人愿意与人联系，经常这样，这“个人”就成了“大人”．

这两幅图告诉我们：当你把自己的命运与全人类的命运紧紧地联系在一起时，你一定会成为一个伟大的人．

你的这种伟大表现在：向往自然与文明，不仅为自己也为他人创造文明、幸福与快乐．

通常的学习过程

n  听老师讲课

n  做练习

n  考前复习

n  考试

在这样的学习过程中：

n  有些人连教科书都没有完整地看一遍．

n  作业做完后连自己也不愿再看

n  考前复习时觉得许多东西好像没有学过

n  考试完了，数学就完了

 

怎样学习数学？

1．培养良好的学习品质

n  对未知世界充满志趣、好奇心，并能持之以恒．

2．选择正确的学习态度

一个真实美好的学习过程，是经历人类文明洗礼，欣赏并遵守规则的过程．

正确的数学学习态度，最重要的一个特征是做好作业．

n  严格按照数学表达规范做作业．

n  做好问题比多做问题重要．

n  做作业不仅仅是做教材、教辅上的习题，更重要的是以学习的内容为例子去体会如何学习．

n  你的作业就是你的作品，就是你在别人面前的“脸面”与表现．

 

3．清楚数学层次

我们吃的大米来自图9中的稻穗．从结构上看，这稻穗有“根”、“茎叶”、“谷壳”、“米”，这很像数学中的概念、原理、方法、思想．

n  数学概念是数学的“细胞”，是数学思维的出发点与落脚点，更是数学思维路径的呈现形式．

n  数学原理是指建立在数学概念上的结论、规律，如公理、定理、公式，等等．

n  数学方法是指运用数学概念与原理去解决具体问题的策略与步骤．如因式分解法、配方法、选定系数法、数学归纳法、比较法、同一法、分析法、综合法、反证法，等等．

n  数学思想是指在整个数学领域具有指导意义的数学精神与灵魂．如数学模型的思想、数形结合的思想、分类讨论的思想、等价转换的思想、归纳与类比的思想、元的思想、算法思想、用样本估计总体的统计思想，等等．

我们平常吃的是米而不是稻草（稻穗的茎叶）．即使这样，我们也是从米中汲取营养后，把剩余物要排泄掉．

同样，我们学习数学，也要学会从数学中汲取营养，这种营养我们把它称为数学素养．

高中阶段的数学核心素养是指数学抽象，逻辑推理，数学建模，运算能力，直观想像，数据分析．

图10反映了数学的层次，也让我们清楚了学习数学的主要目的是汲取数学核心素养．

 

问题3：三角形

n  概念：三角形．

n  原理：内角和定理，正弦定理，余弦定理．

n  方法：解三角形，利用解三角形进行测量．

n  思想：元的思想，并由此认识三角形为何是三元对象，理解三角形全等的判定定理．

n  素养：数学抽象，逻辑推理，数学建模，直观想像．

4．明确数学学习目标

n  概念清

n  原理透

n  方法熟

n  思想通

n  能力强

高中数学中，常见的数学能力有抽象概括能力、运算求解能力、作图能力、推理论证能力、空间想像能力、数据收集与整理能力、数学技术工具使用能力、数学地提出、分析和解决问题的能力，数学表达和交流的能力，独立获取数学知识的能力，等等．

5．理解数学研究过程

n  （1）从大量相关现象中抽象出研究对象，给研究对象下定义．

n  （2）将研究对象表示出来．

n  （3）置新对象于已有的结构中．

n  （4）研究新对象与原有对象的关系，得出新的结论．

n  （5）运用新结论为人类服务．

问题4：城市街道路线问题

如图，棋盘式街道中，大量人流从A地出发随机选择路线到达B地．若限制行进的方向只能向右或向上，那么人流中经过E地的人与选择不经过E地的人哪个多？

（1）定义：大量的合乎条件的行走路线我们定义其为路径．

（2）表示：把横向街道从左至右依次表示为a，b，c，纵向街道从下到上依次表示为1，2，3，这样，任何一条路径就是a，b，c，1，2，3的一个排列．

（3）联系：当我们写出若干个排列后，我们就能发现这些排列的特点：a，b，c的顺序一定，1，2，3的顺序也一定，符合要求的排列与某类组合对应．

（4）结论：经过E点的人要多于不经过E点的人．

（5）应用：如果要减轻E点的交通压力，可以在最靠近A点的两个街口安置信号灯，通过调整不同方向的通行时间来减少通过E点的人流．

6．把书读薄

高中数学学习让你形成的数学思维，概括起来就是“模特元定界”．

元的思想

n  任何事物都有确定它的基本要素，任何事物的变化及事物与事物之间的联系也均由事物的基本要素引起并确定，我们把确定事物的基本要素称为事物的“元”．从元的角度观察、分析和认识事物，探求事物各部分之间的联系，发现事物的变化规律，并最终利用规律去解决问题的基本思想称为元的思想．

7．掌握学习技术

人类文明的进步已深深地影响并改变着人们的生活．

学习技术的诞生，同样改变了人们的思维与学习方式．如今，如果你拥有一台图形计算器，你再也不用花几天时间去完成一个繁杂的计算，甚至于在瞬间就能得到你想要的复杂图象、回归方程、定积分的值、微分方程的解……

同学们在学习时发生困难，大多是因为数学思维的严谨性、概念的抽象性与计算的复杂性，而解决困难的有效方法就是寻求技术的支持，在抽象的概念与直观的情景之间建立“多元联系”．

8．追求简单与普遍性

结语

数学所记录的是人类活动的痕迹，它有温度、情感与生命．

n  只要你运用数学的知识、方法与思想，无论你做什么事情，都会表现出专业的品质．

n  只要你拥有良好的数学素养，无论你的言语、思想，还是行为，都会表现出对规则的尊重和良好的教养．

学习数学的真正目的不是为了考试

学习数学让同学们增长学识，丰富学养，并因此拥有良好的教养．
