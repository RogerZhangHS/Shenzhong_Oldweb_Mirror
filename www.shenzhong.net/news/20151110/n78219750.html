万科梅沙书院首届足球赛圆满结束

深圳中学　2015-11-10　

荣辱与共 搏尽无悔

——万科梅沙书院首届足球赛圆满结束

 

11月5日下午，万科梅沙书院首届足球赛循环至第三轮。今天的天气相比前两次比赛好了许多，清爽舒适取代了炎热，同学们的专注和稳重取代了焦躁与不安。今天的比赛共两场: 4:05-4:50 class 6 and class 3 VS class 2 and class 4，4:55-5:30 class 1 and class 5 VS class 7 and class 8。在前两轮比赛中，一班与五班的组合以全胜战绩傲视群雄，六班、三班与二班、四班的同学都是一胜一负，七班八班则两场遗憾落败。今天是小组循环赛的最后一天，所有队员都等待着爆发的刹那，战况比前两天的激烈许多，各支队伍为大家献上了一场场扣人心弦的对决。

参赛队员合影后便迅速投入比赛。二、四班队员在比赛之初便采取高压攻势，迅速的拿下了一分，三、六班也不甘示弱，组织起反攻，双方实力非常相近，开始了激烈的拉锯战。场中，三、六班女生多次将球攻入禁区内，但被对方后卫与守门员一一化解；第三节比赛一开始，三、六班队员一改原先偏保守的风格，变为积极突进、抢断，但遭到二、四班的反扑，被再入一球，在遗憾中失掉了比赛。

第二场开局，一、五班对战七、八班，比赛的节奏出奇迅速，开场六秒，第一分就被一、五班率先得去，并在第一节内又进两球；而七八班的同学们并不气馁，顽强抵挡对方的进攻；尽管最终以0:3负于对手，但这份执着的拼搏将成为永恒的坚持，感染着集体中的每一份子。

万科梅沙书院的首届足球赛在同学们的欢笑与感动中结束了。最终，一、五班组合以全胜战绩勇夺冠军，二、四班以两胜一负的赛果名列亚军。一班涂家瑞与四班李霁恒分别以7球和4球的成绩位列射手榜前二；一班涂家瑞与二班张鉴力加冕助攻王。比赛会有输赢，但在过程之中每个人获得的经验和知识，才是收获中难能可贵的存在。本届球赛激发了书院同学全民参与体育赛事的意识，提高了他们的身体素质，并通过亲身经历了解了足球比赛的知识，获得了一定的足球技巧。在不同班级混合组队的赛制下，大家不懈磨合，依旧配合的十分默契，培养了团队合作意识，增强了不同班级之间的联系。

这是足球赛结束的一天，却不是同学们的满腔热情结束的时刻。接近冬天，夜幕即将降临的足球场上，我们曾放手一搏、全力以赴，以最高昂的斗志和青春的激情应战；我们胜不骄、败不馁，以最充分的准备和纯粹的努力确证汗水能够承托起梦想的重量。我们的身边有荣辱与共的队友，背后有交付信任的同学，他们捕捉每一个精彩的瞬间，挡住每一次飞出场外的足球，维持每一场赛事的秩序，清理每一回遗落的水瓶与垃圾。我们会记住球场上的同学，他们代表了梅沙学子健康、昂扬的青春，我们更不会忘记默默奉献的同伴，他们身上有梅沙书院倡导的世界公民的素养——关心集体，乐于奉献。我们永远是一个团队，我们为集体而战！

 

撰稿学生：晏子如、周钰涵、王思涵、孙婉婷、徐嘉彤、刘晓易

（万科梅沙书院校园媒体中心·杂志部）

照片来源：万科梅沙书院摄影社（VMA Photography Studio）

指导教师：邬宁馨
